# 用户管理页面 - 需求文档

## 页面概述

用户管理页面是管理员后台的核心模块，用于管理系统用户账号。支持创建用户、编辑用户信息、重置密码、启用/停用账号等功能。该页面采用管理员开通制，确保只有授权人员才能使用系统，保障数据安全。

## 用户角色

| 角色 | 权限说明 |
|------|----------|
| 管理员 | 拥有全部权限：创建/编辑/删除用户、重置密码、启用/停用账号 |
| 采集员 | 无权访问此页面 |

## 功能需求

### 功能1：用户列表展示

- **输入**：无（默认加载全部用户）
- **处理**：
  1. 从 `users` 集合查询所有用户记录
  2. 关联查询用户的采集统计信息（创建机房数、最后登录时间）
  3. 按创建时间倒序排列
  4. 渲染用户列表
- **输出**：用户列表（包含姓名、手机号、角色、状态、创建时间、最后登录时间、操作）

### 功能2：新增用户

- **输入**：用户信息表单（姓名、手机号、初始密码、角色、所属区域）
- **处理**：
  1. 校验手机号格式（11位数字）
  2. 校验手机号是否已存在
  3. 校验密码强度（至少6位，包含数字和字母）
  4. 创建用户记录，状态设为 `pending`（待开通）
  5. 记录操作日志
- **输出**：新用户创建成功提示，用户列表刷新

### 功能3：编辑用户信息

- **输入**：选择用户 → 修改字段（姓名、角色、所属区域）
- **处理**：
  1. 校验必填字段
  2. 更新用户记录
  3. 记录操作日志
- **输出**：用户信息更新成功提示

### 功能4：重置密码

- **输入**：选择用户 → 点击重置密码
- **处理**：
  1. 弹窗提示确认操作
  2. 可选择重置为默认密码或自定义密码
  3. 更新用户密码（加密存储）
  4. 标记需要首次登录修改密码
  5. 记录操作日志
- **输出**：密码重置成功提示，显示新密码

### 功能5：启用/停用账号

- **输入**：选择用户 → 点击启用/停用按钮
- **处理**：
  1. 弹窗确认操作
  2. 显示停用原因输入框（停用时必填）
  3. 更新用户状态（`approved` ↔ `disabled`）
  4. 记录操作日志
- **输出**：状态变更成功提示

### 功能6：删除用户

- **输入**：选择用户 → 点击删除按钮
- **处理**：
  1. 检查用户是否有关联数据（创建的机房记录）
  2. 如有关联数据，提示无法删除或选择数据转移
  3. 如无关联数据，弹窗确认删除
  4. 软删除用户记录（标记 `_deleted: true`）
  5. 记录操作日志
- **输出**：删除成功提示

### 功能7：用户搜索

- **输入**：关键词（姓名、手机号）
- **处理**：
  1. 对姓名和手机号进行模糊匹配
  2. 实时返回搜索结果
- **输出**：匹配的用户列表

### 功能8：用户筛选

- **输入**：筛选条件（角色、状态、所属区域）
- **处理**：
  1. 接收筛选条件
  2. 构建查询，支持多条件组合
  3. 返回筛选结果
- **输出**：符合筛选条件的用户列表

### 功能9：查看用户详情

- **输入**：点击用户记录
- **处理**：
  1. 查询用户完整信息
  2. 统计用户操作数据（创建机房数、编辑次数、最后操作时间）
  3. 查询用户最近操作记录
- **输出**：用户详情面板（基本信息 + 统计数据 + 最近操作）

## 交互流程

```
管理员进入用户管理页面
    ↓
加载用户列表（表格展示）
    ↓
├── [新增用户] → 填写表单 → 校验 → 创建成功
├── [编辑用户] → 选择用户 → 修改信息 → 保存
├── [重置密码] → 选择用户 → 确认 → 设置新密码 → 完成
├── [启用/停用] → 选择用户 → 确认 → 状态变更
├── [删除用户] → 选择用户 → 检查关联 → 确认删除
├── [搜索] → 输入关键词 → 实时搜索
├── [筛选] → 选择条件 → 刷新列表
└── [查看详情] → 点击用户 → 展开详情面板
```

## 字段定义

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| _id | string | 是 | 用户唯一标识（系统自动生成） |
| phone | string | 是 | 手机号（11位，唯一） |
| password | string | 是 | 密码（加密存储） |
| name | string | 是 | 用户姓名 |
| role | string | 是 | 角色：admin / collector |
| status | string | 是 | 状态：pending / approved / disabled / rejected |
| region | string | 否 | 所属区域（如：东南亚、日本、欧洲、澳洲） |
| team | string | 否 | 所属团队（预留扩展字段） |
| avatar | string | 否 | 头像URL |
| last_login_at | datetime | 否 | 最后登录时间 |
| login_count | number | 否 | 登录次数 |
| password_changed | boolean | 否 | 是否已修改初始密码 |
| created_by | string | 否 | 创建人ID（管理员） |
| created_by_name | string | 否 | 创建人姓名 |
| created_at | datetime | 是 | 创建时间 |
| updated_at | datetime | 是 | 更新时间 |
| _deleted | boolean | 否 | 软删除标记 |

### 用户状态枚举

| 状态 | 标识 | 说明 | UI展示 |
|------|------|------|--------|
| 待开通 | `pending` | 已创建，等待使用 | 橙色标签 |
| 已开通 | `approved` | 正常使用 | 绿色标签 |
| 已停用 | `disabled` | 管理员停用 | 红色标签 |
| 已拒绝 | `rejected` | 开通申请被拒绝 | 灰色标签 |

### 角色枚举

| 角色 | 标识 | 说明 |
|------|------|------|
| 管理员 | `admin` | 全部权限，包括用户管理、数据管理 |
| 采集员 | `collector` | 前台采集、查看、编辑、导入导出 |

- [ ] 支持按状态筛选（全部/待开通/已开通/已停用）
- [ ] 新增用户表单校验正常（手机号格式、密码强度）
- [ ] 新增用户时检测手机号是否已存在
- [ ] 编辑用户信息功能正常
- [ ] 重置密码功能正常，可选择默认密码或自定义密码

- [ ] 启用/停用操作后状态实时更新

- [ ] 用户详情显示统计数据（创建机房数、最后操作时间）

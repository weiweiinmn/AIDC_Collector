# 客户管理页面 - 需求文档

## 页面概述

客户管理页面是业务拓展团队管理目标客户信息的核心模块。支持维护客户基本信息、追踪客户跟进状态、管理机房与客户的预留关系。该页面帮助团队清晰地了解每个客户的机房需求意向和跟进进度，实现精准的客户关系管理和商机追踪。

## 用户角色

| 角色 | 权限说明 |
|------|----------|
| 管理员 | 全部权限：创建/编辑/删除客户、分配客户经理、导出客户数据 |
| 采集员 | 查看客户信息、关联机房预留、更新跟进状态（限自己负责的客户） |
| 客户经理 | 读写自己负责客户的跟进信息 |

## 功能需求

### 功能1：客户列表展示

- **输入**：无（默认加载全部客户）
- **处理**：
  1. 从 `clients` 集合查询所有客户记录
  2. 关联查询每个客户的预留机房数量
  3. 按创建时间倒序排列
  4. 渲染客户列表
- **输出**：客户列表（包含客户名称、联系人、联系电话、状态、客户经理、预留机房数、跟进状态）

### 功能2：新增客户

- **输入**：客户信息表单（公司名称、联系人、联系电话、邮箱、所属行业、客户经理、备注）
- **处理**：
  1. 校验必填字段（公司名称、联系人）
  2. 校验电话/邮箱格式
  3. 检查公司名称是否已存在（避免重复）
  4. 创建客户记录，状态默认为 `active`（活跃）
  5. 记录操作日志
- **输出**：新客户创建成功提示，客户列表刷新

### 功能3：编辑客户信息

- **输入**：选择客户 → 修改字段
- **处理**：
  1. 校验必填字段
  2. 更新客户记录
  3. 记录操作日志
- **输出**：客户信息更新成功提示

### 功能4：删除客户

- **输入**：选择客户 → 点击删除按钮
- **处理**：
  1. 检查客户是否有关联的机房预留
  2. 如有预留，提示需先取消关联才能删除
  3. 如无预留，弹窗确认删除
  4. 删除客户记录
  5. 记录操作日志
- **输出**：删除成功提示或删除失败原因

### 功能5：客户状态管理

- **输入**：选择客户 → 更改状态
- **处理**：
  1. 支持状态切换：活跃 ↔ 暂停 ↔ 流失
  2. 暂停/流失时需要填写原因
  3. 更新客户状态
  4. 记录状态变更历史
  5. 记录操作日志
- **输出**：状态变更成功提示

### 功能6：跟进状态更新

- **输入**：选择客户 → 更新跟进状态和备注
- **处理**：
  1. 更新跟进状态（初步沟通/方案已发/谈判中/已签约/已放弃）
  2. 更新跟进备注
  3. 记录跟进时间
  4. 记录操作日志
- **输出**：跟进信息更新成功提示

### 功能7：客户搜索

- **输入**：关键词（公司名称、联系人、联系电话）
- **处理**：
  1. 对多个字段进行模糊匹配
  2. 实时返回搜索结果
- **输出**：匹配的客户列表

### 功能8：客户筛选

- **输入**：筛选条件（状态、跟进状态、客户经理、所属行业）
- **处理**：
  1. 接收筛选条件
  2. 构建查询，支持多条件组合
  3. 返回筛选结果
- **输出**：符合筛选条件的客户列表

### 功能9：查看客户详情

- **输入**：点击客户记录
- **处理**：
  1. 查询客户完整信息
  2. 查询关联的预留机房列表
  3. 查询跟进历史记录
  4. 统计客户数据（预留机房数、意向区域分布）
- **输出**：客户详情面板（基本信息 + 预留机房 + 跟进历史）

### 功能10：预留机房管理

- **输入**：在客户详情页管理预留机房
- **处理**：
  1. 显示当前预留的机房列表
  2. 支持添加新预留（从机房列表选择）
  3. 支持取消预留
  4. 同步更新机房的 `reservedClients` 字段
  5. 记录操作日志
- **输出**：预留关系更新成功

### 功能11：客户数据导出

- **输入**：选择导出范围（全部/当前筛选结果）
- **处理**：
  1. 查询客户数据
  2. 生成 Excel 文件（包含客户基本信息 + 预留机房汇总）
  3. 提供下载链接
- **输出**：可下载的 Excel 文件

## 交互流程

```
用户进入客户管理页面
    ↓
加载客户列表（表格展示）
    ↓
├── [新增客户] → 填写表单 → 校验 → 创建成功
├── [编辑客户] → 选择客户 → 修改信息 → 保存
├── [删除客户] → 选择客户 → 检查关联 → 确认删除
├── [状态管理] → 选择客户 → 更改状态 → 填写原因 → 保存
├── [跟进更新] → 选择客户 → 更新状态/备注 → 保存
├── [搜索] → 输入关键词 → 实时搜索
├── [筛选] → 选择条件 → 刷新列表
├── [查看详情] → 点击客户 → 展开详情面板
│   ├── 查看基本信息
│   ├── 查看预留机房列表
│   ├── 添加/取消预留
│   └── 查看跟进历史
└── [导出] → 选择范围 → 生成Excel → 下载
```

## 字段定义

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| _id | string | 是 | 客户唯一标识 |
| name | string | 是 | 公司名称 |
| short_name | string | 否 | 公司简称 |
| contact_person | string | 是 | 联系人姓名 |
| contact_phone | string | 否 | 联系电话 |
| contact_email | string | 否 | 联系邮箱 |
| industry | string | 否 | 所属行业（互联网/金融/制造/政府/其他） |
| status | string | 是 | 客户状态：active / paused / churned |
| manager_id | string | 否 | 客户经理ID |
| manager_name | string | 否 | 客户经理姓名（冗余存储） |
| follow_status | string | 否 | 跟进状态：initial / proposal / negotiating / contracted / abandoned |
| follow_note | text | 否 | 跟进备注 |
| last_follow_at | datetime | 否 | 最后跟进时间 |
| address | string | 否 | 公司地址 |
| website | string | 否 | 公司网站 |
| notes | text | 否 | 备注 |
| created_by | string | 是 | 创建人ID |
| created_by_name | string | 是 | 创建人姓名 |
| created_at | datetime | 是 | 创建时间 |
| updated_at | datetime | 是 | 更新时间 |
| _deleted | boolean | 否 | 软删除标记 |

### 客户状态枚举

| 状态 | 标识 | 说明 | UI展示 |
|------|------|------|--------|
| 活跃 | `active` | 正在跟进的客户 | 绿色标签 |
| 暂停 | `paused` | 暂时停止跟进 | 橙色标签 |
| 流失 | `churned` | 已放弃的客户 | 红色标签 |

### 跟进状态枚举

| 状态 | 标识 | 说明 |
|------|------|------|
| 初步沟通 | `initial` | 刚接触，了解需求 |
| 方案已发 | `proposal` | 已发送方案/报价 |
| 谈判中 | `negotiating` | 深入谈判阶段 |
| 已签约 | `contracted` | 已签订合同 |
| 已放弃 | `abandoned` | 合作未能达成 |

### 行业枚举

| 行业 | 标识 |
|------|------|
| 互联网 | `internet` |
| 金融 | `finance` |
| 制造 | `manufacturing` |
| 政府 | `government` |
| 其他 | `other` |

# 操作日志页面 - 需求文档

## 页面概述

操作日志页面是系统审计追踪的核心模块，用于记录和展示用户在系统中的所有关键操作行为。该页面为管理员提供完整的操作追溯能力，确保数据安全性和可审计性，支持问题排查和责任认定。

## 用户角色

| 角色 | 权限说明 |
|------|----------|
| 管理员 | 可查看所有用户的操作日志，支持导出日志 |
| 采集员 | 可查看自己的操作记录（仅限于本人操作） |

## 功能需求

### 功能1：操作日志列表展示

- **输入**：无（默认加载最近日志）
- **处理**：
  1. 从 `operationLogs` 集合分页查询操作记录
  2. 按时间倒序排列（最新操作在前）
  3. 关联查询操作人姓名、角色信息
  4. 渲染列表展示
- **输出**：操作日志列表（包含操作人、操作类型、操作对象、操作时间、操作结果）

### 功能2：操作日志筛选

- **输入**：筛选条件（操作人、操作类型、时间范围、操作对象类型）
- **处理**：
  1. 接收用户选择的筛选条件
  2. 构建查询条件，支持多条件组合筛选
  3. 重新查询数据库
  4. 返回筛选结果
- **输出**：符合筛选条件的操作日志列表

### 功能3：操作日志搜索

- **输入**：关键词（支持机房名称、操作人姓名、备注内容）
- **处理**：
  1. 接收用户输入的搜索关键词
  2. 对关键字段进行模糊匹配查询
  3. 返回匹配结果
- **输出**：匹配关键词的操作日志列表

### 功能4：操作日志详情查看

- **输入**：点击某条日志记录
- **处理**：
  1. 展开详情面板或跳转详情页
  2. 显示完整的操作信息，包括：
     - 操作前数据快照（before）
     - 操作后数据快照（after）
     - 变更字段明细
     - IP 地址、设备信息（如可获取）
- **输出**：完整的操作详情视图

### 功能5：操作日志导出

- **输入**：选择导出范围（全部/当前筛选结果）+ 导出格式
- **处理**：
  1. 根据选择范围查询日志数据
  2. 生成 Excel 文件
  3. 提供下载链接
- **输出**：可下载的 Excel 文件

## 交互流程

```
用户进入操作日志页面
    ↓
默认加载最近 100 条操作日志
    ↓
展示日志列表（卡片/表格形式）
    ↓
├── [筛选] → 选择筛选条件 → 刷新列表
├── [搜索] → 输入关键词 → 实时搜索
├── [查看详情] → 点击日志 → 展开详情面板
├── [导出] → 选择导出范围 → 生成Excel → 下载
└── [分页] → 滚动加载/点击翻页
```

## 字段定义

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| log_id | string | 是 | 日志唯一标识 |
| operator_id | string | 是 | 操作人用户ID |
| operator_name | string | 是 | 操作人姓名（冗余存储） |
| operator_role | string | 是 | 操作人角色（admin/collector） |
| operation_type | string | 是 | 操作类型（见操作类型枚举） |
| target_type | string | 是 | 操作对象类型（datacenter/user/client） |
| target_id | string | 否 | 操作对象ID |
| target_name | string | 否 | 操作对象名称（机房名称/用户姓名/客户名称） |
| operation_detail | object | 否 | 操作详情（JSON格式，包含变更内容） |
| before_value | object | 否 | 操作前数据快照 |
| after_value | object | 否 | 操作后数据快照 |
| changed_fields | array | 否 | 变更字段列表 |
| operation_result | string | 是 | 操作结果（success/failed） |
| error_message | string | 否 | 失败原因（操作失败时记录） |
| ip_address | string | 否 | 操作IP地址 |
| user_agent | string | 否 | 客户端信息 |
| created_at | datetime | 是 | 操作时间（服务端时间） |

### 操作类型枚举

| 操作类型 | 标识 | 说明 |
|----------|------|------|
| 创建机房 | `datacenter_create` | 新增机房记录 |
| 编辑机房 | `datacenter_update` | 修改机房信息 |
| 删除机房 | `datacenter_delete` | 删除机房记录 |
| 导入机房 | `datacenter_import` | Excel导入机房 |
| 导出机房 | `datacenter_export` | 导出机房数据 |
| 用户登录 | `user_login` | 用户登录系统 |
| 用户登出 | `user_logout` | 用户退出登录 |
| 修改密码 | `user_password_change` | 用户修改密码 |
| 创建客户 | `client_create` | 新增客户 |
| 编辑客户 | `client_update` | 修改客户信息 |
| 删除客户 | `client_delete` | 删除客户 |
| 预留分配 | `reservation_assign` | 机房预留客户分配 |
| 预留取消 | `reservation_cancel` | 取消机房预留 |

## P1 - 中等问题（影响体验）

### ISSUE-004: Dashboard 缺少核心指标卡片
- **文件**: dashboard-v2.html
- **问题**: 
  - 缺少「客户总数」卡片
  - 缺少「设备总数」卡片
  - 缺少「告警数量」卡片（红色高亮）
- **建议**: 添加这 3 个指标卡片，告警数量需特殊样式处理

### ISSUE-005: Dashboard 交互效果缺失
- **文件**: dashboard-v2.html
- **问题**: 
  - 无骨架屏加载效果
  - 数字无计数器动画
  - 无自动刷新功能（需求要求 60 秒）
- **建议**: 
  - 添加骨架屏组件
  - 实现数字从 0 到目标值的动画（500ms ease-out）
  - 添加 setInterval 实现 60 秒自动刷新

### ISSUE-006: 机房详情页缺少核心功能区域
- **文件**: datacenter-detail-v2.html
- **问题**: 以下区域完全缺失
  - 机房基本信息展示（名称/编码/城市/地址等）
  - 机柜列表区域
  - 设备列表区域
  - 操作记录区域
  - 编辑/删除操作按钮
- **建议**: 参考 PAGE-datacenter.md 补充所有功能区域

### ISSUE-007: 权限控制未实现
- **文件**: dashboard-v2.html, datacenter-detail-v2.html
- **问题**: 页面未根据用户角色（admin/manager/viewer）显示不同内容
- **需求**: 
  - viewer: 只读，无操作按钮
  - manager: 可编辑，不可删除
  - admin: 完全权限
- **建议**: 添加角色判断逻辑，根据角色显示/隐藏操作按钮

## 2. 页面定位与功能概述

机房管理是 AIDC Collector 的核心模块，用于管理所有机房的档案信息、机柜配置、设备台账。运维人员通过此模块实现对机房资产的集中管理、状态监控和运维操作。

## 6. 交互流程

### 6.1 新建机房

```
点击「新建机房」→ 填写表单 → 实时校验 → 点击保存
  → 成功：跳转列表并高亮新增行，Toast「创建成功」
  → 失败：留在表单，滚动到错误字段
```

### 6.2 编辑机房

```
点击「编辑」→ 跳转 /datacenter/:id/edit
  → 加载已有数据到表单
  → 修改字段 → 保存 → 跳转详情页，Toast「保存成功」
  → 取消：返回详情页，数据不变
```

### 6.3 删除机房

```
点击「删除」→ 弹出确认框
  → 确认：调用删除 API → 跳转列表，Toast「删除成功」
  → 取消：关闭确认框，不操作
  ⚠️ 有设备关联时，提示「该机房下存在 X 台设备，请先移除后再删除」
```

### 6.4 查看详情

```
点击「查看」或点击行 → 跳转 /datacenter/:id
  → 加载详情数据
  → 左侧基本信息 + 右侧关联数据（机柜/设备/日志）
```

## 7. 验收标准

1. ✅ 列表默认按创建时间倒序排列
2. ✅ 搜索防抖300ms，避免频繁请求
3. ✅ 删除前校验关联数据，给出明确提示
4. ✅ 详情页左侧固定基本信息，右侧可滚动
5. ✅ 所有时间格式统一：YYYY-MM-DD HH:mm
6. ✅ 列表每页默认20条，支持10/20/50/100切换
7. ✅ 权限：viewer 角色编辑/删除按钮不可见
8. ✅ 机房编码唯一，重名时表单提示「机房编码已存在」

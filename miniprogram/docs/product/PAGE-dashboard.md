- **路由**：`/dashboard`
- **相关文档**：
  - [PRODUCT-STRUCTURE.md](./PRODUCT-STRUCTURE.md) — 整体架构
  - [COMPONENT-sidebar.md](./COMPONENT-sidebar.md) — 左边栏组件
  - [COMPONENT-table.md](./COMPONENT-table.md) — 表格组件

---

## 2. 页面定位与功能概述

仪表盘是用户登录后的默认首页（首页），提供全局业务数据的实时概览，帮助用户快速掌握平台整体运营状态。

核心价值：
- 一眼看清关键业务指标
- 快速发现异常数据（告警）
- 一键跳转到详情页面

---

## 3. 用户角色与权限

| 角色 | 访问权限 |
|------|----------|
| admin | 完全访问，可查看所有统计 |
| manager | 完全访问 |
| viewer | 只读访问，无法操作任何区块 |

---

## 4. 功能清单

### P0 - 核心指标卡片（必须）

| 功能 | 说明 |
|------|------|
| 机房总数 | 展示当前系统注册的机房数量 |
| 客户总数 | 展示当前客户数量 |
| 设备总数 | 展示所有机房的设备总台数 |
| 告警数量 | 展示当前活跃告警数量（红色高亮） |

### P1 - 图表区

| 功能 | 说明 |
|------|------|
| 机房分布地图 | 中国地图热力图，展示机房地域分布 |
| 设备增长趋势 | 近30天设备入库折线图 |
| 机房状态分布 | 饼图/环形图，正常/异常/维护比例 |
| 客户续费日历 | 月历视图，标注即将到期客户 |

### P2 - 快捷入口 & 待办

| 功能 | 说明 |
|------|------|
| 快捷操作 | 最近操作的机房、客户入口 |
| 待办提醒 | 设备巡检到期、合同到期等提醒 |
| 最新操作日志 | 最近5条操作记录，点击跳转操作日志页 |

---

## 5. 字段定义

### 5.1 指标卡片字段

| 字段 | 数据类型 | 格式 |
|------|----------|------|
| label | string | 中文名称 |
| value | number | 整数，千分位格式化 |
| unit | string | 单位（个、台、人） |
| trend | enum | up / down / flat（相对上周期） |
| trend_value | string | 变化量，如 "+12%" |
| link | string | 点击跳转路由 |

### 5.2 操作日志字段

| 字段 | 数据类型 | 格式 |
|------|----------|------|
| operator | string | 操作人姓名 |
| action | string | 操作类型 |
| target | string | 操作对象（机房名称/客户名称） |
| time | datetime | YYYY-MM-DD HH:mm |

---

## 6. 交互流程

### 6.1 页面加载

1. 用户进入页面 → 显示骨架屏
2. 并行请求所有指标 API
3. 渲染卡片（计数器动画，从0增长到目标值）
4. 渲染图表

### 6.2 数据刷新

- 页面打开时自动刷新一次
- 每60秒自动刷新（可关闭）
- 手动刷新：点击右上角刷新按钮
- 下拉刷新（移动端）

### 6.3 指标卡片点击

- 点击指标卡片 → 跳转对应列表页（带默认筛选条件）
- 例：点击「告警数量」卡片 → 跳转机房列表，筛选状态=告警

---

## 7. 验收标准

1. ✅ 首次加载骨架屏，无白屏
2. ✅ 所有卡片数字从0动画增长到目标值（500ms ease-out）
3. ✅ 告警数量 > 0 时卡片边框高亮红色
4. ✅ 图表可切换时间范围（近7天/30天/90天）
5. ✅ 响应式：移动端卡片改为2列或1列布局
6. ✅ 无数据时显示「--」而非「0」或空
7. ✅ 权限正确：viewer 角色无操作按钮

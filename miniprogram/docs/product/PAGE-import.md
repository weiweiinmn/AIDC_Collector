## 2. 页面定位与功能概述

数据导入模块用于批量导入机房、客户、设备等数据，支持 Excel 模板下载、在线预览、自动校验和导入结果反馈。适用于系统初始化数据迁移和日常批量更新。

## 4. 功能清单

### P0

| 功能 | 说明 |
|------|------|
| 选择导入类型 | 下拉选择：机房 / 客户 / 设备 |
| 下载模板 | 一键下载对应类型的 Excel 导入模板 |
| 上传文件 | 支持 .xlsx / .xls / .csv，上传后自动解析 |
| 数据预览 | 解析后显示前20行，支持翻页预览 |
| 字段校验 | 空值/格式/唯一性校验，错误行高亮标记 |
| 执行导入 | 确认无误后执行导入，实时进度条 |
| 导入报告 | 显示成功数量、失败数量、失败原因明细 |

### P1

| 功能 | 说明 |
|------|------|
| 跳过重复数据 | 勾选后自动跳过已存在的记录（按唯一字段判断） |
| 覆盖已有数据 | 勾选后用导入数据覆盖已有记录 |
| 导入历史 | 查看历史导入记录及结果 |
| 部分导入 | 允许用户手动选择跳过错误行，只导入正确数据 |

### P2

| 功能 | 说明 |
|------|------|
| 定时导入 | 配置定时导入任务（对接数据源） |
| 导入数据回滚 | 导入后可一键回滚（仅限最近一次） |

## 5. 交互流程

### 5.1 完整导入流程

```
① 选择导入类型（机房/客户/设备）
   ↓
② 下载模板 → 填写 Excel → 保存
   ↓
③ 上传文件（拖拽或点击上传）
   → 支持 .xlsx / .xls / .csv，最大 10MB
   ↓
④ 数据解析 + 预览（显示前20条）
   → 自动校验：必填项、格式、唯一性
   → 错误行标红，悬浮显示错误原因
   ↓
⑤ 选择导入策略（跳过重复 / 覆盖 / 部分导入）
   ↓
⑥ 点击「确认导入」
   → 显示进度条（百分比 + 已处理/总数）
   → 后台分批导入（每批100条）
   ↓
⑦ 导入完成 → 显示报告
   → 成功：绿色，显示数量
   → 失败：红色，显示失败列表（可下载失败行明细）
```

### 5.2 部分导入

```
预览阶段 → 勾选「部分导入」
  → 错误行前显示复选框（默认不勾选）
  → 用户手动取消勾选错误行
  → 点击确认 → 仅导入勾选行
```

## 6. 字段定义

### 6.1 导入类型配置

| 导入类型 | 唯一标识字段 | 关联实体 |
|----------|------------|---------|
| 机房 | code（机房编码） | 机房管理 |
| 客户 | code（客户编码） | 客户管理 |
| 设备 | serial_number（序列号） | 设备管理 |

### 6.2 机房导入模板字段

| 字段 | 必填 | 校验规则 |
|------|------|----------|
| code | ✅ | 不超过20字符，唯一 |
| name | ✅ | 不超过50字符 |
| city | ✅ | 下拉选项或文本 |
| address | ✅ | 不超过200字符 |
| contact_name | — | 不超过30字符 |
| contact_phone | — | 手机号格式 |
| area | — | 正数 |
| remark | — | 不超过500字符 |

### 6.3 导入报告字段

| 字段 | 说明 |
|------|------|
| total_count | 总行数 |
| success_count | 成功行数 |
| fail_count | 失败行数 |
| fail_list | 失败行详情，含行号、错误原因 |
| duration | 导入耗时 |
| operator | 操作人 |
| imported_at | 导入时间 |

## 7. 验收标准

1. ✅ 单次导入最多支持 5000 条数据
2. ✅ 支持文件大小不超过 10MB
3. ✅ 校验实时反馈，错误行数 ≤ 100 时全部显示；> 100 时显示前100条并提示
4. ✅ 导入过程可取消（正在处理的批次完成后停止）
5. ✅ 导入完成后自动记录到操作日志
6. ✅ 失败行可下载（包含行号、原始数据、错误原因）
7. ✅ 相同文件二次上传自动提示「是否覆盖上次导入结果」
8. ✅ 导入进度实时更新，页面刷新不丢失进度
9. ✅ 所有导入操作均有权限控制，非授权用户不可访问

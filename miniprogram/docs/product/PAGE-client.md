## 2. 页面定位与功能概述

客户管理模块用于管理所有客户/租户信息，记录客户与机房的租赁关系、机柜占用情况和合同信息。销售和运营人员通过此模块跟踪客户资产分布和业务状态。

## 6. 交互流程

### 6.1 新建客户

```
点击「新建客户」→ 填写基本信息 + 联系人 → 提交
  → 成功：跳转客户详情页，Toast「客户创建成功」
  → 失败：表单错误提示
```

### 6.2 关联机房

```
客户详情页 → 点击「关联机房」→ 弹窗选择已有机房
  → 选择机房 + 指定机柜 → 确认关联
  → 关联成功：刷新机柜列表
```

### 6.3 合同管理

```
客户详情页 → 点击「合同管理」→ 进入合同 Tab
  → 新增合同：填写合同编号、起止日期、金额
  → 续约：自动带入旧合同信息，修改日期
  → 终止：二次确认，更新状态为 expired
```

### 6.4 删除客户

```
点击「删除」→ 检测是否有关联机房/设备
  → 有关联：提示「该客户存在 X 个机房关联，请先解除关联」
  → 无关联：二次确认 → 删除 → 跳转列表
```

## 7. 验收标准

1. ✅ 客户名称支持模糊搜索（≥2字符触发）
2. ✅ 合同到期30天内自动标红「即将到期」标签
3. ✅ 合同已过期客户置灰显示
4. ✅ 删除客户前强制校验关联关系
5. ✅ 客户编码唯一性校验
6. ✅ 主要联系人字段有且仅有一个（is_primary = true）
7. ✅ 导出 Excel 包含所有列表字段
8. ✅ 权限：viewer 角色所有操作按钮不可见

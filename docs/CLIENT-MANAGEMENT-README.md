## 功能实现

### 1. 客户管理页面 ✅

#### 侧边栏入口
- 在"数据中心"分组下新增"客户管理"菜单项
- 显示客户总数徽章 (navCustomers)

#### 客户列表
- 表格展示: 客户名称、联系人、联系电话、状态、预留机房数、操作
- 支持搜索(名称/联系人)
- 支持状态筛选(活跃/暂停/流失)
- 显示每个客户预留的机房数量

#### 客户CRUD操作
- **新增客户**: 弹窗表单,包含名称、联系人、电话、状态
- **编辑客户**: 复用新增弹窗,预填充现有数据
- **删除客户**: 
  - 检查是否有关联机房预留
  - 如有预留,提示先取消预留
  - 确认后删除

#### 客户状态
- 活跃 (active): 绿色徽章
- 暂停 (paused): 橙色徽章
- 流失 (churned): 红色徽章

### 2. 机房预留功能 ✅

#### 机房表单 - 预留客户字段
- 实现多选下拉组件 (multi-select)
- 仅显示"活跃"状态的客户
- 支持搜索和快速选择
- 已选客户显示为标签,可单独移除

#### 机房列表 - 预留客户显示
- 表格新增"预留客户"列
- 以标签形式显示多个客户
- 无预留显示"-"

#### 筛选器 - 按客户筛选
- 筛选栏新增"客户筛选"下拉
- 选择客户后,仅显示该客户预留的机房
- 支持与其他筛选条件组合

#### 数据关联
- 机房数据结构新增 `reservedClients` 数组字段
- 存储客户ID列表
- 支持一个机房预留给多个客户

### 3. 数据结构 ✅

```javascript
// 客户对象
{
  _id: 'c1',           // 唯一ID
  name: '腾讯云',      // 客户名称
  contact: '张经理',   // 联系人
  phone: '13800138000', // 联系电话
  status: 'active'     // 状态: active/paused/churned
}

// 机房对象(新增字段)
{
  ...,
  reservedClients: ['c1', 'c2']  // 预留客户ID数组
}
```

## 技术实现

### 多选下拉组件
- 自定义CSS组件,无外部依赖
- 支持点击展开/收起
- 已选项显示为可移除标签
- 支持点击外部自动关闭

### 数据流
1. 全局状态管理
   - `allCustomers`: 所有客户数据
   - `allDatacenters`: 所有机房数据
   - `selectedClients`: 当前选中的客户ID列表

2. 数据同步
   - 新增/编辑机房时保存 `reservedClients`
   - 删除客户前检查关联机房
   - 客户状态变更实时更新多选下拉

### UI/UX优化
- 客户标签使用蓝色系,与状态徽章区分
- 多选下拉支持键盘操作
- 删除操作有二次确认
- 预留客户数实时计算显示

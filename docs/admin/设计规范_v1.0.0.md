# AIDC Collector — 统一设计规范
> 版本：v1.0.0 | 日期：2026-04-08 | 制定：小创 🎨

---

## 一、设计原则

1. **统一性**：所有页面使用同一设计语言，无主题突变
2. **可扩展**：CSS变量驱动，便于主题切换
3. **渐进增强**：基础功能优先，装饰性动效锦上添花
4. **无障碍**：颜色对比度 ≥ 4.5:1，支持键盘操作

## 三、字体系统

### 3.1 主字体：Inter（推荐）
统一所有页面使用 **Inter** 作为UI字体。

```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
```

### 3.2 字体层级

| 级别 | 样式 | 用途 |
|-----|------|------|
| `font-display` | 32px / 700 / -0.02em | 大标题（页面标题） |
| `font-h1` | 24px / 700 / -0.02em | 一级标题 |
| `font-h2` | 20px / 600 / -0.01em | 二级标题（卡片标题） |
| `font-h3` | 16px / 600 | 三级标题（字段标签） |
| `font-body` | 14px / 400 | 正文 |
| `font-sm` | 13px / 400 | 辅助文字 |
| `font-xs` | 12px / 500 | 标签/徽章/小字 |
| `font-mono` | JetBrains Mono / 14px | 代码/ID/技术数据 |

## 四、侧边栏规范

### 4.1 结构
```
宽度: 240px (固定)
背景: var(--primary) 纯色 或 渐变
Logo区: 高度64px, 左侧logo + 右侧产品名
导航区: padding 12px, 每项高度44px, 圆角8px
用户区: 高度72px, 底部固定, 头像+姓名+角色
```

### 4.2 导航项状态

| 状态 | 样式 |
|-----|------|
| 默认 | 背景透明, 文字 rgba(255,255,255,0.72) |
| 悬停 | 背景 rgba(255,255,255,0.08), 文字 #fff |
| 选中 | 背景 rgba(255,255,255,0.15), 左侧4px橙色竖条 |
| 子菜单展开 | 箭头旋转180°, 子菜单高度展开 |

### 4.3 子菜单样式
```css
.nav-submenu {

  padding-left: 44px;  /* 缩进对齐 */

  max-height: 200px;
}
.nav-submenu-item a {
  height: 40px;
  padding: 0 12px;
  color: rgba(255,255,255,0.5);
  font-size: 13px;
}
.nav-submenu-item a:hover,
.nav-submenu-item.active a {
  color: #fff;
```

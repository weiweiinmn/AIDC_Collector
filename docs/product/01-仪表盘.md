# 仪表盘页面产品文档

**页面**: `dashboard-v2.html`  
**版本**: v2.0  
**日期**: 2026-04-15

---

## 页面概述

仪表盘是系统的首页，展示机房数据的整体概览和关键指标，帮助用户快速了解业务状态。

---

## 功能模块

### 1. KPI 指标卡片区

**位置**: 页面顶部  
**布局**: 4列网格 (grid-template-columns: repeat(4, 1fr))

#### 指标卡片

| 卡片 | 指标值 | 副标题 | 颜色 |
|------|--------|--------|------|
| 机房总数 | stats.total | 共 X 个机房 | 绿色 #22c55e |
| 本周新增 | stats.weekCount | 本月累计 X 个 | 蓝色 #3b82f6 |
| KYC未完成 | undone | 已完成 X · 未完成 X | 橙色 #f59e0b |
| 待KYC | not_started | 待处理 X 个 | 紫色 #8b5cf6 |

#### 数据计算逻辑

```javascript
// 机房总数
kpi-total-value = stats.total

// 本周新增
kpi-today-value = stats.weekCount
kpi-today-sub = stats.monthCount

// KYC未完成
pendingKYC = not_started + in_progress + pending_doc
kpi-kyc-value = pendingKYC
kpi-kyc-sub = approved + not_required

// 待KYC
kpi-neg-value = not_started
```

### 2. 状态分布进度条

**位置**: KPI区下方  
**功能**: 展示各状态机房的数量分布和占比

#### 状态映射

```javascript
var STATUS_MAP = {
    'new':         { label:'新增',   cls:'info'    },
    'collected':   { label:'已收录', cls:'success' },
    'negotiating': { label:'洽谈中', cls:'warning' },
    'signed':      { label:'已签约', cls:'accent'  },
    'rejected':    { label:'已淘汰', cls:'danger'  },
};
```

#### 展示规则
- 按数量降序排列
- 显示状态标签、数量、百分比
- 进度条宽度 = (count/total * 100).toFixed(1)
- 无数据时显示"暂无数据"

### 3. 国家分布面板

**布局**: 垂直列表，最多显示6个国家

## 数据加载流程

```javascript
function loadDashboard() {
    // 1. 初始化种子数据
    AIDC_API.seedDatacenters();
    AIDC_API.seedUsers();
    
    // 2. 获取统计数据
    var stats = AIDC_API.getStats().data;
    
    // 3. 渲染KPI卡片
    renderKPICards(stats);
    
    // 4. 渲染状态分布
    renderStatusBars(stats.statusDist, stats.total);
    
    // 5. 加载完整列表
    var list = AIDC_API.getDatacenters({ pageSize: 1000 }).data.list;
    
    // 6. 渲染国家分布
    renderCountry(list, stats.total);
    
    // 7. 渲染KYC分布
    renderKYC(list);
    
    // 8. 渲染最近录入
    renderRecent(list.slice(0, 6));
}
```

## API 接口

### AIDC_API.getStats()

**返回数据**:
```javascript
{
  code: 0,
  data: {
    total: 16,           // 机房总数
    weekCount: 2,        // 本周新增
    monthCount: 5,       // 本月新增
    statusDist: {        // 状态分布
      new: 3,
      collected: 8,
      negotiating: 4,
      signed: 1
    }
  }
}
```

### AIDC_API.getDatacenters(params)

**参数**:
```javascript
{
  pageSize: 1000,    // 每页数量
  page: 1            // 页码
}
```

## 空状态处理

当无数据时，各模块显示:
- KPI卡片: 显示 "—"
- 状态分布: "暂无数据"
- 国家分布: "暂无数据"
- KYC分布: 正常显示，数量为0
- 最近录入: "暂无数据" + 提示文字

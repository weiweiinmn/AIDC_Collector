# 添加机房页面产品文档

**页面**: `datacenter-form.html`  
**版本**: v2.0  
**日期**: 2026-04-15

---

## 页面概述

添加机房页面用于录入新的机房信息，支持完整的机房数据字段录入，包括基础信息、机柜资源、电力系统、制冷系统、网络接入、认证配套等多个模块。

### 2. 基础信息模块

**卡片标题**: 基础信息  
**图标**: 信息图标 (蓝色)

#### 字段清单

| 字段名 | 字段ID | 类型 | 必填 | 选项/说明 |
|--------|--------|------|------|-----------|
| 机房名称 | dc-name | 文本 | ✓ | 最大100字符，如"AIMS 吉隆坡" |

function addCustomer() {
    var input = document.getElementById('customer-input');
    var name = input.value.trim();
    if (!name) return;
    
    // 创建标签
    var tag = document.createElement('span');
    tag.className = 'customer-tag';

    el.parentElement.remove();

### 10. 表单验证

#### 必填字段验证

```javascript
function validateForm() {
    var errors = [];
    
    // 机房名称必填
    var name = document.getElementById('dc-name').value.trim();
    if (!name) {
        errors.push('机房名称不能为空');
    }
    
    // 状态必填
    var status = document.querySelector('[name="status"]').value;
    if (!status) {
        errors.push('请选择机房状态');
    }
    
    // 数值字段范围验证
    var pue = document.querySelector('[name="pue"]').value;
    if (pue && (parseFloat(pue) < 1.0 || parseFloat(pue) > 3.0)) {
        errors.push('PUE值应在1.0-3.0之间');
    }
    
    return errors;
}
```

#### 提交处理

```javascript
function submitForm() {
    // 1. 验证
    var errors = validateForm();
    if (errors.length > 0) {
        alert('请完善以下信息:\n' + errors.join('\n'));
        return;
    }
    
    // 2. 收集数据
    var data = collectFormData();
    
    // 3. 调用API保存
    var result = AIDC_API.createDatacenter(data);
    
    if (result.code === 0) {
        alert('保存成功！');
        window.location.href = 'datacenter-list.html';
    } else {
        alert('保存失败: ' + result.message);
    }
}
```

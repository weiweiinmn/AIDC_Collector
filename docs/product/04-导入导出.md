## 页面概述

导入导出页面提供机房数据的批量导入和导出功能，支持Excel(.xlsx)、CSV(.csv)、JSON(.json)等多种格式，并提供标准模板下载。

#### 2.1 上传区

**功能**: 拖拽或点击上传文件

**支持多文件**: 可同时选择多个文件

**交互**:

#### 2.2 文件列表

上传后显示文件列表:

```javascript
var selectedFiles = [];

function handleFiles(files) {
    for (var i = 0; i < files.length; i++) {
        var f = files[i];
        var ext = f.name.split('.').pop().toLowerCase();
        
        // 格式验证
        if (['xlsx', 'xls', 'csv'].indexOf(ext) === -1) {
            alert('不支持的格式：' + f.name);
            continue;
        }
        
        selectedFiles.push(f);
    }
    renderFileList();
    updateImportButton();
}
```

**文件项展示**:
- 文件类型图标 (.xlsx绿色, .csv蓝色)
- 文件名 (超长截断显示省略号)
- 文件大小
- 删除按钮

#### 2.3 导入操作

**按钮状态**:
- 无文件时: 禁用状态
- 有文件时: 可点击

**导入流程**:
```javascript
function doImport() {
    if (selectedFiles.length === 0) return;
    
    var results = {
        success: 0,
        failed: 0,
        errors: []
    };
    
    selectedFiles.forEach(function(file) {
        var ext = file.name.split('.').pop().toLowerCase();
        
        try {
            var data;
            if (ext === 'csv') {
                data = parseCSV(file);
            } else {
                data = parseExcel(file);
            }
            
            // 数据验证和导入
            data.forEach(function(row) {
                var validation = validateRow(row);
                if (validation.valid) {
                    AIDC_API.createDatacenter(row);
                    results.success++;
                } else {
                    results.failed++;
                    results.errors.push({
                        file: file.name,
                        row: row,
                        errors: validation.errors
                    });
                }
            });
        } catch (e) {
            results.failed++;
            results.errors.push({
                file: file.name,
                error: e.message
            });
        }
    });
    
    showImportResult(results);
}
```

### 3. 导出数据模块

#### 3.1 统计概览

**4格统计**:
| 统计项 | 字段 | 说明 |
|--------|------|------|
| 总数 | totalCount | 机房总数量 |
| 新建 | newCount | status=new 的数量 |
| 洽谈中 | negotiatingCount | status=negotiating 的数量 |
| 已收录 | completedCount | status=collected 的数量 |

```javascript
function loadStats() {
    var res = AIDC_API.getDatacenters({ pageSize: 1000 });
    var list = res.data ? res.data.list || [] : [];
    
    document.getElementById('totalCount').textContent = list.length;
    document.getElementById('newCount').textContent = list.filter(d => d.status === 'new').length;
    document.getElementById('negotiatingCount').textContent = list.filter(d => d.status === 'negotiating').length;
    document.getElementById('completedCount').textContent = list.filter(d => d.status === 'collected').length;
}
```

#### 3.2 导出选项

**导出格式** (单选):
- Excel (.xlsx) - 默认选中
- CSV (.csv)
- JSON (.json)

**导出范围** (单选):
- 全部 - 导出所有机房数据
- 当前筛选 - 导出当前筛选条件下的数据 (需配合列表页筛选)

#### 3.3 导出流程

```javascript
function doExport() {
    // 获取导出格式
    var format = document.querySelector('input[name="export-format"]:checked').value;
    var scope = document.querySelector('input[name="export-scope"]:checked').value;
    
    // 获取数据
    var data;
    if (scope === 'all') {
        var res = AIDC_API.getDatacenters({ pageSize: 1000 });
        data = res.data ? res.data.list || [] : [];
    } else {
        // 获取当前筛选条件的数据 (需从列表页传递)
        data = getFilteredData();
    }
    
    // 根据格式导出
    var filename = '机房数据_' + formatDate(new Date(), 'YYYYMMDD');
    
    switch (format) {
        case 'xlsx':
            exportToExcel(data, filename + '.xlsx');
            break;
        case 'csv':
            exportToCSV(data, filename + '.csv');
            break;
        case 'json':
            exportToJSON(data, filename + '.json');
            break;
    }
    
    showExportResult({
        success: true,
        count: data.length,
        filename: filename + '.' + format
    });
}
```

#### 3.4 导出格式说明

**Excel (.xlsx)**:
- 包含表头
- 自动列宽
- 状态/KYC等字段使用颜色标注
- 最适合查看和打印

**CSV (.csv)**:
- 纯文本格式
- 逗号分隔
- UTF-8编码
- 通用性强，可用Excel打开

**JSON (.json)**:
- 完整数据结构
- 包含所有字段
- 适合程序处理和数据备份

#### 4.2 模板字段

模板包含以下标准字段:

```

基础信息:
- 国家 (country)
- 省/州 (province)
- 城市 (city)
- 详细地址 (address)
- 信息来源 (source)
- 服务商名称 (service_provider)
- 产权归属 (ownership)
- 预计交付时间 (expectedDate)

规模:
- 总机柜数量 (rackCount)
- 可用机柜数量 (rackAvailable)
- 单机柜功率kW (powerPerRack)
- 总电力MW (mw)
- 可用电力MW (availPowerMw)
- 可扩容电力MW (expandablePowerMw)

电力系统:
- 市电总容量kVA (mainsCapacityKva)
- 变压器配置 (transformerConfig)
- 柴发冗余等级 (gensetRedundancy)
- UPS配置 (upsConfig)
- 电力SLA (powerSla)

制冷系统:
- 制冷方式 (coolingSystem)
- 液冷支持 (liquidCoolingSupport)
- PUE值 (pue_value)
- 地板承重 (floorLoad)

网络:
- 网络路由数 (networkRoutes)
- 运营商 (isp)
- 800G支持 (support800g)

认证:
- 等级认证 (levelCertification)
- ISO认证 (isoCertified)

联系:
- 客户名称 (customer)
- 联系人 (contact)
- 联系电话 (contactPhone)
- 对接同事 (liaison)

其他:
- KYC状态 (kycStatus)
- 备注 (remark)
```

#### 4.3 下载模板

```javascript
function downloadTemplate(format) {
    var templateData = generateTemplateData();
    var filename = '机房导入模板';
    
    if (format === 'xlsx') {
        exportToExcel(templateData, filename + '.xlsx');
    } else if (format === 'csv') {
        exportToCSV(templateData, filename + '.csv');
    }
}

function generateTemplateData() {
    // 生成带示例数据的模板
    return [{
        name: '示例机房名称',
        country: '新加坡',
        status: 'new',
        // ... 其他示例字段
    }];
}
```

#### 4.4 使用说明

```
使用说明:
1. 下载模板文件
2. 按模板格式填写数据
3. 上传填好的文件导入

注意事项:
- 机房名称和状态为必填项
- 状态值必须是: new, collected, negotiating, signed, rejected
- KYC状态值必须是: not_started, in_progress, pending_doc, approved, rejected, not_required
- 日期格式建议: YYYY-MM-DD
```

## 错误处理

### 导入错误类型

| 错误码 | 说明 | 处理建议 |
|--------|------|----------|
| INVALID_FORMAT | 文件格式不支持 | 请上传.xlsx或.csv文件 |
| MISSING_REQUIRED | 缺少必填字段 | 检查机房名称和状态是否填写 |
| INVALID_STATUS | 状态值无效 | 使用标准状态值 |
| INVALID_KYC | KYC状态值无效 | 使用标准KYC状态值 |
| PARSE_ERROR | 文件解析失败 | 检查文件是否损坏 |
| DUPLICATE_NAME | 机房名称重复 | 修改名称或更新现有记录 |

### 导出错误处理

- 无数据时: 提示"暂无机房数据可导出"
- 导出失败: 提示"导出失败，请重试"

# AIDC Collector 管理后台 - 产品结构文档

## 1. 文档概述

### 1.1 文档目的

本文档定义了 AIDC Collector 管理后台的整体产品架构、导航结构、通用设计原则，是整个产品文档体系的总纲。所有子文档均以本文档为顶层入口。

### 1.2 产品定位

AIDC Collector 是一款面向 IDC（互联网数据中心）运营方的资产采集与管理平台，核心用于跟踪和管理机房设备、客户服务、团队协作及数据流转。

### 1.3 目标用户

| 角色 | 描述 | 可见模块 |
|------|------|----------|
| admin | 超级管理员 | 全部模块（含管理员设置） |
| manager | 运营经理 | 仪表盘、机房管理、客户管理、用户管理（查看）、操作日志、数据导入/导出、系统设置（个人设置） |
| viewer | 普通查看者 | 仪表盘、机房管理（只读）、客户管理（只读）、操作日志（只读） |

## 4. 通用设计原则

### 4.1 布局规范

- **整体布局**：左侧固定宽度左边栏（约240px）+ 右侧自适应内容区
- **内容区**：最大宽度 1400px，水平居中，内边距 24px
- **响应式**：1024px 以下边栏可折叠为图标模式，768px 以下切换为汉堡菜单

### 4.2 颜色系统

| 用途 | 色值 |
|------|------|
| 主色 | #2563EB |
| 成功 | #10B981 |
| 警告 | #F59E0B |
| 危险 | #EF4444 |
| 文字主色 | #111827 |
| 文字次要 | #6B7280 |
| 边框 | #E5E7EB |
| 背景 | #F9FAFB |
| 卡片背景 | #FFFFFF |

### 4.3 字体系统

- 主字体：`-apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif`
- 标题字重：600~700
- 正文字重：400
- 基准字号：14px

### 4.4 组件规范引用

| 组件 | 规范文档 |
|------|----------|
| 左边栏 | [COMPONENT-sidebar.md](./COMPONENT-sidebar.md) |
| 表格 | [COMPONENT-table.md](./COMPONENT-table.md) |
| 表单 | [COMPONENT-form.md](./COMPONENT-form.md) |

### 4.5 权限控制原则

- 前端根据用户角色动态渲染菜单项，未授权菜单不展示
- 所有后端 API 均需权限校验，前端不做为安全边界
- 操作日志记录所有写操作（含操作人、时间、变更前后内容）

## 5. 状态设计

### 5.1 通用状态

| 状态 | 样式 |
|------|------|
| 正常/启用 | 绿色标签 |
| 禁用/停用 | 灰色标签 |
| 告警/异常 | 红色标签 |
| 待处理/进行中 | 蓝色标签 |

### 5.2 分页

- 默认每页 20 条，可选 10/20/50/100
- 显示总数、分页导航（首页/上一页/页码/下一页/末页）

---

## 6. 错误处理

- 表单校验：字段级实时校验 + 提交前整体校验
- API 错误：Toast 提示，描述性文案，不暴露内部错误码
- 空状态：显示友好插图 + 操作引导文案
- 网络错误：重试按钮 + 当前状态保留
